# CFVanguard Pack Simulator
Projeto independente de um simulador de pacotes para o TCG Cardfight Vanguard

Novas cartas podem ser adicionadas ao simulador ao colocar arquivos .csv na pasta packs

Como utilizar:

1 - python instalado

2 - abra a pasta pelo cmd

3 - abra o arquivo rodar.py

4 - escolha qualquer uma das opções, elas terão instruções para como prosseguir
